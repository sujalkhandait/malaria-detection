# 🦠 Malaria Detection using Deep Learning

This project is a deep learning based malaria cell image classification system built using Flask and TensorFlow.

## Features
- Upload blood cell images
- Detect malaria infection
- Deep learning powered prediction
- Responsive web interface

## Tech Stack
- Python
- Flask
- TensorFlow / Keras
- HTML/CSS
- Docker

## Model
The model classifies images into:
- Parasitized
- Uninfected

## Deployment
Hosted on Hugging Face Spaces using Docker.
